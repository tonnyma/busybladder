zGlossary - Example of showing terms only once

This example demonstrates the 'showonce' option and will only create a link for the first term found on a page, in this case our company name.

Script

$(document).ready(function () {
	$('body').glossary('terms.json', {
		showonce: true
	});
});

Result

A heading showing the definition of Zazar in a pop-up

The Zazar definition can be shown within a paragraph.
